Ship reward expiry warnings in 6 TDD tasks across 2 repos.
Daily cron in CasaPerks-API reads the rewards table, batches users with rewards expiring in exactly 3 days, calls the existing notification service. The web client reads the same expires_at field and renders an ExpiryBanner on rewards within 7 days; expired rewards move to a collapsed "Expired" section.
